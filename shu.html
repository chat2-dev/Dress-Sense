How it works
  • Core shape: single SVG path for crisp edges and scalability.
  • Depth: gradients + drop shadows + subtle fold paths.
  • Tactile feel: CSS radial gradients and a tiny noise overlay emulate knit texture.
Quick edits
  • Change the color: modify --fabric-main, --fabric-dark and --fabric-light in the top of the CSS.
  • Make it longer/shorter: adjust the main SVG path coordinates for the body.
